## Overview

DisplayBuddy Advanced is an external display control application that adjusts brightness, contrast, volume, and input source of external monitors using the DDC/CI protocol, bringing the same display controls to external monitors that macOS provides for built-in screens. It solves the frustration of reaching for physical monitor buttons to adjust brightness or volume by providing software controls through keyboard shortcuts, menu bar sliders, and automated schedules.

DDC/CI control communicates directly with external monitors through the display connection to adjust brightness, contrast, volume, and input source without physical button interaction. The control works with most monitors connected via HDMI, DisplayPort, or USB-C that support the DDC/CI standard.

Keyboard shortcut control maps brightness and volume adjustment to configurable keyboard shortcuts that work identically to the built-in display brightness keys, making external monitor adjustment as natural as built-in display control.

Multi-display management handles multiple external monitors independently with per-display settings, enabling different brightness levels for different monitors and synchronized adjustment across all displays.

For users with external monitors who want software control over display settings without reaching for physical buttons, DisplayBuddy Advanced is the display control tool that makes external monitors as controllable as built-in screens.

## Performance

DisplayBuddy balances performance with system resource usage intelligently. Operations are handled in the background, keeping your system responsive.

Apple Silicon acceleration is fully utilized on M-series chips. On Intel machines, hardware acceleration is used wherever available.
